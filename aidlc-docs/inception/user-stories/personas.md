# User Personas

## Persona 1: 고객 (Customer)

| 항목 | 내용 |
|---|---|
| 이름 | 테이블 고객 |
| 역할 | 매장 방문 고객 |
| 사용 디바이스 | 테이블 태블릿 (모바일 웹 브라우저) |
| 목표 | 대기 없이 빠르고 편리하게 메뉴를 주문하고, 주문 상태를 확인 |
| 특성 | 별도 로그인 없이 즉시 사용, 터치 기반 인터랙션, 식사 중 추가 주문 가능 |
| 기술 수준 | 다양 (디지털 기기에 익숙하지 않은 사용자도 포함) |
| 핵심 니즈 | 직관적 UI, 빠른 메뉴 탐색, 간편한 주문, 인원수 기반 메뉴 추천 |

## Persona 2: 매장 관리자 (Admin)

| 항목 | 내용 |
|---|---|
| 이름 | 매장 관리자 |
| 역할 | 매장 운영 담당자 |
| 사용 디바이스 | 데스크톱 PC/노트북 (웹 브라우저) |
| 목표 | 실시간 주문 모니터링, 효율적 테이블 관리, 메뉴 관리 |
| 특성 | 사전 등록된 계정으로 로그인, 장시간 세션 유지, 다수 테이블 동시 관리 |
| 기술 수준 | 중간 (기본적인 웹 애플리케이션 사용 가능) |
| 핵심 니즈 | 실시간 주문 알림, 한눈에 보는 테이블 현황, 빠른 주문 상태 변경, 결제 완료 처리 |
